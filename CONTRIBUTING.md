# Contributing to dspy-community

Thanks for your interest in the DSPy community! There are several ways to contribute.

## Add your library to the meta-package

If you've built a pip-installable library that extends DSPy, we'd love to include it.

**Requirements:**
- Published on PyPI
- Works with `dspy>=2.6`
- Has a README with usage examples
- Has tests
- Actively maintained

**How to submit:**
1. Open an issue with the title `[New Library] your-package-name`
2. Include: PyPI link, GitHub repo, short description, and why it's useful for DSPy users
3. We'll review and add it as an optional extra

## Add your project to the showcase

Built something cool with DSPy? We'll link it in the README.

**How to submit:**
1. Open an issue with the title `[New Project] your-project-name`
2. Include: GitHub repo, short description

## Add a language port

Porting DSPy to a new language? Let us know and we'll feature it.

## Report issues

If you find issues with the meta-package itself (dependency conflicts, etc.), please open an issue.
